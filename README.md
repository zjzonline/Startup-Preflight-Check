## 这是什么？

一个开源的 AI Agent Skill（Hermes Agent 生态），帮助创业者在投入时间、金钱、辞职之前，系统检查项目能否**安全、合规、低成本**启动和上线。

**不是商业计划书写手，不是鸡汤导师，是开工前风险检查员。**

---

## 它能做什么？

- 🩺 **咨询式分轮问诊** — 像创业顾问一样一对一聊，每次只问1-2个关键问题
- 🔍 **七大维度风险扫描** — 资质备案、平台审核、资金成本、资源缺口、城市适配、获客难度
- 🚦 **continue / pause / stop 决策** — 明确告诉你现在该做什么、不该做什么
- 📋 **19项结构化报告** — 含成功标准、停止条件、官方核实渠道
- 🗓️ **7天最小验证路径** — 3,000元即可完成首轮测试
